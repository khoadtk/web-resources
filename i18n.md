# i18n (Đa ngôn ngữ)

Cho web hỗ trợ nhiều ngôn ngữ (vd Việt / Anh).

## Khái niệm

- **i18n** (internationalization) — chuẩn bị code để dịch được.
- **l10n** (localization) — bản dịch + định dạng thực tế (ngày, tiền, số).
- **Locale** — mã ngôn ngữ/vùng: `vi`, `en`, `en-US`.
- Chuỗi dịch để trong file **JSON theo từng ngôn ngữ**, code chỉ tham chiếu **key**.

## Thư viện

- **[next-intl](https://next-intl.dev)** — i18n cho Next.js App Router (khuyên dùng)
- **[react-i18next](https://react.i18next.com)** — phổ biến nhất cho React
- **[FormatJS / react-intl](https://formatjs.io)** — chuẩn ICU message
- **[Paraglide](https://inlang.com/m/gerre34r/library-inlang-paraglide-js)** — i18n type-safe, nhẹ

## Cách dùng

### HTML tĩnh

Tự map key → chuỗi bằng JS, đổi text theo lựa chọn:

```js
const dict = {
  vi: { hello: 'Xin chào' },
  en: { hello: 'Hello' },
};
let lang = 'vi';
document.querySelectorAll('[data-i18n]').forEach(el => {
  el.textContent = dict[lang][el.dataset.i18n];
});
```

```html
<h1 data-i18n="hello">Xin chào</h1>
```

### React

**react-i18next**:

```bash
npm install i18next react-i18next
```

```jsx
import { useTranslation } from 'react-i18next';
function Title() {
  const { t, i18n } = useTranslation();
  return (
    <>
      <h1>{t('hello')}</h1>
      <button onClick={() => i18n.changeLanguage('en')}>EN</button>
    </>
  );
}
```

### Next.js

**next-intl** — thường dùng route theo locale (`/vi/...`, `/en/...`):

```jsx
// messages/vi.json → { "hello": "Xin chào" }
import { useTranslations } from 'next-intl';
function Title() {
  const t = useTranslations();
  return <h1>{t('hello')}</h1>;
}
```

## Mẹo

- **Không hard-code chuỗi** trong JSX — luôn qua key ngay từ đầu (thêm ngôn ngữ sau rất mệt nếu để trễ).
- Định dạng ngày/số/tiền bằng **`Intl`** có sẵn của trình duyệt:

```js
new Intl.NumberFormat('vi-VN', { style: 'currency', currency: 'VND' }).format(1000000);
```

- Lưu ngôn ngữ user chọn vào cookie/localStorage để nhớ lần sau.
- Xem thêm [dark-mode.md](dark-mode.md) — cùng dạng "chuyển chế độ toàn app".
